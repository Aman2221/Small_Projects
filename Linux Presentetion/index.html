HOW TO CREATE PARTATION IN LINUX ?

A ) There are two ways of creating partition is linux

  1. 1. Using graphical user interface.
  2. 2. Using command line interface.

B ) Types of partition

  1. 1. Primary/Physical partition.
  2. 2. Extended partition.
  3. 3. Logical partition.

C ) Explanation

  1. Primary : Can only create four partitions.
  2. Extended : Can not create file System.
  3. Logical : Crated using Extended partition.
    Maximum 11 logical partition

D ) Partition Commands

  1. m : Help
  2. p : Primary
  3. e : Extended
  4. n : New
  5. d : Delete
  6. w : Save

E ) Creating Partition

  1. Get root access
  2. fdisk - l (exsisting files)
  3. fdisk /dev/sda
  4. n (new partition)
  5. Choose type
  6. P (primary) or E (extended)
  7. Specify Space
  8. w (save Partitions)
